ChainKeeper Bot
===============

Description
-----------
ChainKeeper Bot is a Telegram bot designed to help track the condition of your bicycle chain. It integrates with the Strava API to fetch data on the distance traveled and notifies the user when it's time to clean and lube the chain.

Features
--------
- Track the condition of your bicycle chain.
- Notifications for when to clean and lube the chain.
- Integration with Strava API to fetch distance data.
- Survey functionality to gather user input on chain maintenance intervals.
